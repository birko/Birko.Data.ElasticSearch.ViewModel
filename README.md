# Birko.Data.ElasticSearch.ViewModel

Elasticsearch ViewModel repository implementations for the Birko Framework.

## Features

- ViewModel repositories for Elasticsearch
- Document count and cache clearing
- Custom search capabilities

## Installation

```bash
dotnet add package Birko.Data.ElasticSearch.ViewModel
```

## Dependencies

- Birko.Data.Core (AbstractModel, ViewModels)
- Birko.Data.Stores (store interfaces)
- Birko.Data.Repositories (repository interfaces)
- Birko.Data.ViewModel
- Birko.Data.ElasticSearch
